# Task Manager

## Description
This project uses a Python programme to manage tasks and users. It was designed for the Capstone Project of the HyperionDev Data Science course.

## Features:
- User log in, registration and authentication
- View existing tasks
- Add and assign new tasks
- Delete tasks
- Generate task and user overview reports
- Display statistics on users and task completion

## Files:
- task_manager.py: Python programme
- tasks.txt: text file containing details of tasks
- user.txt: text file containing details of users
